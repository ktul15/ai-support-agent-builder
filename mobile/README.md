# mobile (Flutter consumer app)

The customer-facing assistant: streaming chat, tappable citations, and the
distinct "I don't know" refusal state.

**Not yet initialized.** Scaffolded as a placeholder in issue #2; the Flutter
app (BLoC + Dio + auto_route) is created in Phase 3 (issue #34 onward).

This machine uses **FVM** — prefer `fvm flutter` / `fvm dart`, and run
`fvm dart run build_runner build -d` after editing `@freezed` / `@injectable` /
`auto_route` files. Not part of the Node workspaces.
